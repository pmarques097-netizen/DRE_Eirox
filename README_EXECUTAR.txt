PROJETO EIrox DRE ONLINE

Para executar local:
1) Abra o CMD na pasta do projeto.
2) Instale dependências: pip install -r requirements.txt
3) Execute: streamlit run app.py

Para subir online no Streamlit Cloud:
1) Envie estes arquivos para um repositório GitHub.
2) No Streamlit Cloud, escolha app.py como arquivo principal.
3) Mantenha o Excel em data/DRE_Consolidado_Moderno.xlsx ou use upload pelo painel.

Estrutura:
- app.py
- requirements.txt
- assets/logo_eirox.png
- data/DRE_Consolidado_Moderno.xlsx
- .streamlit/config.toml
